Welcome To The Parkourii Server Website

What is this?

Parkourii is a minecraft server, that you can join using the ip: Parkourii.Minehut.GG on Java 1.17.1 . It is a parkour server with multiple levels, checkpoints and a special rank and name tag colour in tab when you finish. There are 4 stages (gates) and each one is increasing in difficulty.
